<product_requirements>
The core requirement was a web app for annotating de-identified health records from CSV, featuring user authentication, structured tag annotation (SDOH domains, valence, skip), metrics (IRR, prevalence), and admin functions (CSV upload/delete, user management, export).

Key features added throughout development:
- Resources Page: Admin upload, all user download, with preview for PDFs/images, search, filter, pagination.
- Home Page: Quick links, default landing.
- Bulk Deletion: Users, documents, annotations.
- Account Management: Change password, update display name.
- Message Board (UI present, functionality not detailed in trajectory).
- Enhanced Analytics: Per-user stats, sentences left, IRR, server-rendered category/project charts.
- Subject-Based Annotation: Detect subject IDs, prev/next buttons.
- Document/Annotation Export: Original, inline annotated CSV (per-tag rows, detailed columns), annotated paragraphs (reconstructs original text with inline tags), per-user downloads.
- Default Project Management: Admin sets a global project name, new uploads use it, bulk reassign existing documents.
- Annotation UI: Individual tag toggling, valence styling (green/red), keyboard shortcuts, Save/Save+Prev/Save+Next buttons, auto-advance on skip, toast notifications, global confirmation dialogs.
</product_requirements>
